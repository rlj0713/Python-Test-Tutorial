# Simple Calculator – pytest Learning Project

[Pytest Help](https://youtu.be/NinrbvXj8i4?si=Of7UUUCCf-X-xuI1)

This is a small Python calculator built specifically to help students learn how to write and run tests using pytest.

The goal of this project is not to build a production calculator, but to understand:
- how functions are tested
- how pytest detects failures
- how to think about edge cases
- how to improve code through testing

---

# What This Project Does

This app provides a simple command-line calculator with basic operations:

- Addition
- Subtraction
- Multiplication
- Division
- Power (exponentiation)

Each operation is implemented as a standalone function so it can be tested independently.
